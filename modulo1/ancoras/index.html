adicionamos um target="blank", isso significa que o link será aberto em uma nova guia, ou pagina em branco, sem fechar o de origem, vejamos todas:

Devemos indicar para qual site ou de quem pertence o link que estamos indicando atraves de:

  • next- o link é proxima parte do documento
  • prev- o link é parte anterior do documento
  • author- o link é de outra pessoa que escreveu o site
  • external- o link faz parte de outro site que não faz parte do atual
  • nofollow- indica que o link é um site pago
