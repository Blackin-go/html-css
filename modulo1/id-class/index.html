Pseudo-elemento CSS

Os pseudo elementos agem como um filho do elemento, alteram parte do elemento, diferente do pseudo classe que altera completamente o elemento, como a cor por exemplo. sempre depis de 2x dois pontos ( :: ), veja abaixo.

    Teste peseudo elemento

  • ::after
  • ::first-letter
  • ::first-line
