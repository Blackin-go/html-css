eSCOPOS DE GRUPOS

Há duas tags de definição para colunas e linhas, semanticamente explicando o objetivo daquela coluna, é o que chamamos de escopos e as tags são:
scope="col" --> para colunas
scope="row" --> para linhas
scope="colgroup" --> para grupo de colunas
scope="rowgroup" --> para grupo de linhas
