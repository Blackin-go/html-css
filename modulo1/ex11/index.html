Pseudo-classes CSS

Pseudo-classes é uma palavra chave adicionada às declarações de um seletor após um sinal de dois pontos e especificam um estado especial de um elemento, Veja alguns tipos de pseudo-classes

  • : hover
  • : visited
  • : active
  • : checked
  • : empty
  • : focus

Pseudo-elemento CSS

Palavra chave adicionada às declarações de um seletor após dois sinais de pontos e permitem que você formate um pedaço específico do elemento referenciado. Veja alguns tipos de pseudo-elementos:

  • ::before
  • ::after
  • ::first-letter
  • ::first-line
  • ::
